Instruções de Uso

Bem vindo(a) a Biblioteca Digital. Siga as instruções corretamente:

  • Aproxime o código de barras do livro até a câmera de maneira que fique bem evidente o código de barras
  • Em seguida, capture a imagem corretamente
  • Se a imagem não estiver boa e nitida, tente novamente!
  • Se a imagem capturada estiver bem evidente envie a imagem
  • Se a imagem falhar, você pode tentar também enviando o arquivo da imagem salva nesse sistema!
